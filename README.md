# SplitCam
Projeto que deixa a camera em modo picture in picture.

## Motivação
Este projeto foi desenvolvido com um propósito simples e prático: eu sentava de costas para a frente da sala e me sentia incomodado por não conseguir ver quem estava entrando e saindo. O aplicativo funciona como um "retrovisor" na tela, permitindo que eu sempre tenha visibilidade da porta!
